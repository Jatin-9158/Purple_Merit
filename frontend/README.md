# Frontend - User Management System

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env` file (copy from `.env.example`)

3. Start development server:
   ```bash
   npm start
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Features

- Login/Signup pages
- Admin Dashboard with user management
- User Profile page
- Protected routes
- Responsive design
